* **3 Layouts × 4 Positions**: Bar, Box, or Modal — all fully responsive.
* **Live Design Preview**: See changes in real-time in the WordPress admin before saving.
* **Consent Analytics**: Visual dashboard showing acceptance rates and top visitor countries.
* **WCAG 2.1 AA Accessible**: Focus traps, Escape key dismissal, and full screen-reader support.
* **Zero Dependencies**: Pure vanilla JavaScript under 4KB. Won't slow your site down.

## 📦 Installation
1.  Download the latest [cookiezu.zip](https://github.com/flyzal/CookiEzu/releases).
2.  In WordPress, go to **Plugins > Add New > Upload Plugin**.
3.  Upload the ZIP and **Activate**.
4.  Configure your banner in **CookiEzu > Settings**.
