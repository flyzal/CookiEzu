\## üõ†Ô∏è How Can I Contribute?



\### Reporting Bugs

\* Use the \[GitHub Issues](https://github.com/flyzal/CookiEzu/issues) page.

\* Include your WordPress and PHP versions.

\* Describe the exact steps to reproduce the bug.



\### Suggesting Enhancements

\* Open an issue with the "enhancement" label.

\* Explain why the feature would be useful for most users.



\### Pull Requests

We use the \*\*fork-and-pull\*\* model:

1\. \*\*Fork\*\* the repo and create your branch from `main`.

2\. If you've added code that should be tested, add tests!

3\. Ensure your code follows WordPress PHP Coding Standards.

4\. Issue a pull request against the `main` branch.
